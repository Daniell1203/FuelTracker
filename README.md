# FuelTracker

Projeto simples em Python para controle de abastecimento e consumo de combustível.

## Funcionalidades

- Registrar abastecimentos
- Ver histórico
- Calcular consumo médio
- Menu interativo no terminal

## Tecnologias

- Python 3

## Melhorias futuras

- Salvar dados em JSON
- Interface gráfica
- Relatórios
- Gráficos de consumo
